## Setup

Using [cargo](https://doc.rust-lang.org/cargo/)

```
cargo install snakers
```

## Usage

Use arrow keys (or `hjkl` if you prefer the vi key binging) to navigate.

`q` or `Esc` to exit the game.

`r` to play again.
